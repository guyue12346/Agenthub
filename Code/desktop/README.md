# AgentHub 桌面端

AgentHub 桌面端是 Web 工作台之外的独立入口，面向私有部署、局域网访问和长期驻留使用场景。它以 Electron 承载正式 Web 应用，同时补充原生连接设置和桌面端生命周期管理。

## 产品定位

桌面端不是一个独立分叉版本，而是 AgentHub 多端体系中的桌面容器。核心业务能力仍来自正式后端和 Web 工作台，桌面端负责让用户以更接近原生应用的方式进入系统。

## 关键能力

- 独立应用入口，适合固定工作站、内网环境和长期运行场景。
- 原生连接设置，用于选择要访问的 AgentHub 服务地址。
- 复用 Web 工作台能力，包括消息、Hub、工作空间、Agent 状态和后台监控。
- 与浏览器访问保持同一套账号、会话、数据和权限体系。

## 设计取舍

桌面端保持轻容器策略，不在 Electron 层重复实现业务逻辑。这样可以让 Web、桌面端和后端演进保持一致，也减少多端功能分裂。
